LINUX

نظام لينكس Linux هو نواة Kernel لنظام تشغيل حر ومفتوح المصدر وهو يمثل العمود الفقري لمعظم الخوادم، الأنظمة السحابية، وحتى الهواتف الذكية أندرويد في الأصل، لينكس هو "نواة" Kernel طوّرها لينوس تورفالدس عام 1991. ومع دمجها بأدوات مشروع "جنو" GNU ، أصبح لدينا نظام تشغيل متكامل. أهم ما يميزه أنه مفتوح المصدر Open Source ، مما يعني أن أي شخص يمكنه رؤية الكود البرمجي و تعديله وتوزيعه.

إدارة الملفات والمجلدات (File and folder management)

عرض الملفات

يعرض الملفات داخل المجلد الحالي 

ls

يعرض التفاصيل الحجم + الصلاحيات 

ls -l

يعرض المجلدات بشكل شجرة 

tree

التنقل

الدخول إلى مجلد 

cd Documents

الرجوع للخلف 

cd ..

معرفة المسار

يظهر مكانك الحالي 

pwd

إنشاء ملف

ينشئ ملف جديد 

touch file.txt

إنشاء مجلد

ينشئ مجلد باسم 

mkdir test

حذف ملف 

rm file.txt

حذف مجلد فارغ 

rmdir test1

حذف مجلد مع محتوياته 

rm -r test1

نسخ ملف 

cp file1.txt copy.txt

نقل أو إعادة تسمية

تغيير الاسم 

mv file1.txt newname.txt

نقل الملف إلى مجلد 

mv file1.txt test/

يبحث عن ملف داخل المجلد الحالي 

find . -name "file1.txt"

البحث السريع

بحث سريع قد يحتاج تحديث قاعدة البيانات 

locate file1.txt

تغيير الصلاحيات

يعطي كل الصلاحيات 

chmod 777 file2.txt

عرض وقراءة الملفات (View and read files)

اضافة محتوى للملف

اضافة محتوى للملف 

echo "Hello Linux" > file2.txt

عرض محتوى الملف

يعرض كل محتوى الملف دفعة واحدة 

cat file2.txt

عرض تدريجي

يسمح لك تقرأ الملف صفحة صفحة استخدم:↑ ↓ للتنقلq للخروج 

less file2.txt

عرض بسيط

مثل less لكن أقل مرونة 

more file2.txt

أول سطور

عرض أول سطر

head file2.txt

عرض اول سطرين 

head -n 2 file2.txt

اخر سطور

يعرض آخر أسطر 

tail file2.txt

عرض اخر سطرين

tail -n 2 file2.txt

يراقب الملف لحظة بلحظة مثل logs متابعة مباشرة 

tail -f file2.txt

البحث داخل ملف

يبحث عن كلمة داخل الملف 

grep "Mubarak" file2.txt

البحث داخل الملفات والنظام (Searching within files and the system)

يبحث عن كلمة بدون حساسية لحالة الأحرف HELLO = hello

grep -i "mubarak" file2.txt

يبحث داخل كل المجلدات

grep -r "password"

لبحث عن كل ملفات txt

find /home -name "*.txt"

البحث عن الملفات فقط

find . -type f

البحث عن المجلدات فقط

find . -type d

أسرع لأنه يعتمد على قاعدة بيانات

locate file2.txt

معرفة مكان الأمر

which python
الصلاحيات والمستخدمين (Permissions and Users)

المستخدمين

معرفة المستخدم الحالي

whoami

عرض المستخدمين المتصلين

who

انشاء مستخدم جديد

sudo useradd mubarak

إعطاء كلمة مرور

sudo passwd mubarak

حذف مستخدم

sudo userdel mubarak2

التبديل بين المستخدمين

su ali1

المجموعات

إضافة مستخدم إلى مجموعة

sudo usermod -aG developers ali1

الصلاحيات Permissions

كل ملف له 3 أنواع صلاحيات:

r = قراءة

w = كتابة

x = تنفيذ

ls -l

تنقسم إلى:

المالك

المجموعة

الآخرين

تغيير الصلاحيات chmod

يعطي كل الصلاحيات للجميع

chmod 777 file2.txt

قراءة وكتابة للمالك + قراءة فقط للآخرين

chmod 644 file2.txt

صلاحية قراءة للآخرين

chmod 644 test.pdf

عطاء صلاحية كتابة للآخرين

chmod o+w test.pdf

إعطاء صلاحية التنفيذ

chmod o+x test.pdf

إزالة صلاحية التنفيذ

chmod o-x test.pdf

ما هو Networking؟

يختبر الاتصال بالسيرفر إيقاف: Ctrl + C

يوضح مسار الاتصال خطوة خطوة

traceroute google.com

فحص DNS nslookup

يحول الدومين إلى IP

معلومات الشبكة ifconfig

مثل ip a قديم

ifconfig

فحص المنافذ netstat

يعرض المنافذ المفتوحة

البديل الحديث ss

سرع من netstat

ss -tuln

تحميل من الإنترنت wget

wget http://example.com/file.txt

إرسال طلب HTTP curl

curl google.com

nmap google.com

nmap -sV -A example.com

فحص موقع whatweb

يعرف تقنية الموقع PHP, WordPress…

فحص الأدلة dirb

يبحث عن ملفات مخفية
